I have learned a diverse set of languages and frameworks. PHP and Laravel enable server-side web development, while JS, HTML, and CSS are essential for client-side programming. WordPress helps create dynamic websites, and MySQL allows efficient database management. Lastly, React enhances the development of interactive user interfaces.

Projects worked on

Library Management System

The library management system is a Web application that I developed as my final year college project. I took on the challenge of creating the entire project on my own, with some assistance from the internet when needed. To build this system, I utilized various technologies such as PHP for the backend logic, MYSQL for the database management, and HTML, CSS, JavaScript, and Bootstrap for the user interface design.

Online Import System

During my time as a PHP developer at Welpel Info Company, I developed a product import platform that streamlined the process and included features such as product upload, order placement, order approval, secure payment integration, reporting, and other essential functionalities. This project enhanced my skills and contributed to the company's success.

WordPress-Laravel Code Canyon Project

In addition to my experience at Welpel Info Company, I also worked as a freelance WordPress developer on Upwork. One of my notable projects on CodeCanyon involved setting up and validating a payment gateway on a WordPress site. This payment gateway allowed users to purchase a service related to a Laravel project. Additionally, I was responsible for setting up the Laravel project itself, ensuring its proper functioning and integration with the WordPress site. This freelance experience allowed me to further expand my skills in WordPress development and Laravel integration.
